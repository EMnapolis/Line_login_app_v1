หมวดหมู่ เช่น ความหลากหลายของสินค้า, การบริการของพนักงาน ฯลฯ, รีวิวเชิงลบ
ข้อความ Feedback
ชื่อสาขา

🎯 เป้าหมาย:
สร้างรายงานแบบกระชับ โดยรวมข้อมูลทั้งหมดไว้ในไฟล์ Excel ไฟล์เดียว ((ชื่อไฟล์เก่าบางส่วน)_Report_วันที่ประชุม.xlsx) แยกเป็น 7 ชีต เพื่อความสะดวกในการประชุมผู้บริหาร

📋 ข้อกำหนดการจัดทำรายงาน:

1. หัวรายงาน:
ระบุหัวข้อ:
- วันที่ประชุม
- ข้อมูลจาก
- ช่วงเปรียบเทียบข้อมูลซ้ำ
ใช้หัวข้อ #, ## แบบ Markdown (เฉพาะใน Sheet Report และ Report_Summary)

2. รายงานแยกเป็น 7 ชีต:
📄 Sheet 1: Data
คงข้อมูลต้นฉบับทั้งหมดไว้โดยไม่กรอง ID หรือช่วงวันที่

📄 Sheet 2: Report
- สรุป Feedback ทั้งหมด แยกตาม “ประเภทของความคิดเห็น” → “หมวดหมู่”
- ใช้ Markdown เช่น `# ประเภทของความคิดเห็น`, `## หมวดหมู่`
- แต่ละข้อความให้อยู่ในรูปแบบ `- ID xxx: ข้อความ`
- เว้นบรรทัดด้วย `---` ระหว่างหมวดหมู่
- หัวชีตมีหัวข้อ: `# รายงานความคิดเห็นลูกค้า`, `## วันที่ประชุม: ...`, `## ข้อมูลจาก: ...`, `## ช่วงเปรียบเทียบข้อมูลซ้ำ: ...`

📄 Sheet 3: สรุปคำติ และ แนะนำ
- เฉพาะประเภท: “คำติ”, “ข้อเสนอแนะ”, หรือคำเทียบเท่า (ยึดชื่อจากไฟล์)
- แยกตาม “ชื่อสาขา” และ “หมวดหมู่”
- รูปแบบ Markdown:
```
# สาขา
## หมวดหมู่
ID xxx: ข้อความ
---
```
- หากไม่มี ให้ใส่ “ไม่มีคำติหรือคำแนะนำที่โดดเด่น”

📄 Sheet 4: สรุปแต่ละสาขา
- Pivot Table: สาขาเป็นแถว, หมวดหมู่เป็นคอลัมน์
- รวม Total ทั้งในแถวและคอลัมน์

📄 Sheet 5-7:
- แยกแต่ละประเภทไว้ในชีตของตัวเอง
- ใช้ชื่อ Sheet ตาม “ประเภทของความคิดเห็น” ตรงตัว (ห้ามเปลี่ยน)

📘 การตีความ:
- ห้ามแปลหรือดัดแปลงคำพูดลูกค้า
- ห้ามรวม Feedback ข้ามสาขา
- ห้ามแปลชื่อ Sheet หรือรวม Sheet ต่างประเภทเข้าด้วยกัน
- ห้ามใช้ Emoji
- ต้องใส่ ID ครบทุกข้อความ
- ไฟล์ที่ส่งออกต้องเป็น .xlsx และ UTF-8
